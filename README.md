# Atlas of the Web

A human-curated, open-source map of useful websites.

This repository contains structured data only.
The website(s) built on top of this repo render and explore this data.

## What belongs here
- Useful websites
- Non-spammy
- Safe for work by default
- Things you would genuinely recommend to a friend

## What does NOT belong here
- SEO farms
- Affiliate spam
- NSFW / illegal content
- “My new AI tool” with no real value

## How to contribute
1. Add a website entry to an appropriate category file
2. Open a pull request
3. An automated review checks quality, safety, and categorization

That’s it.
